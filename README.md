# Тестовое задание
## Задача
реализовать сервис Balancer.

## Контекст
В приложении присутствуют следующие сущности:

### Процесс:

- требуемое количество памяти,
- требуемое количество процессора;

### Рабочая машина:

- общее количество памяти,
- общее количество процессора;

## Возможности сервиса
Сервис должен обладать следующими возможностями:

- принимать запрос на размещение процесса
- принимать запрос на удаление процесса
- принимать запрос на добавление рабочей машины
- принимать запрос на удаление рабочей машины
- распределять процессы между рабочими-машинами, обеспечивая равномерную нагрузку, выполнять ребалансировку, в случае добавления/удаления машин/процессов
- процесс должен быть размещён полностью на одной машине
- показывать текущее состояние сервиса

## Дополнительная информация
* Написанный функционал покрыть тестами
* Сервис должен запускаться в docker
* Список технологий к использованию
- PHP 7+
- Symfony
- Mysql / Redis или любое иное персистентное хранилище.
